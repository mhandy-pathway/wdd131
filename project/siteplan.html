Site Plan for W06 Project

Site Name: Robi

I chose Robi because I think it could be the name of a robot that could be a personal assistant. My idea is that this site is almost like an AI personal assistant that will help you manage your goals and tasks.

Site Purpose

The purpose of the site is to provide access to free tools to allow users to manage goals (both long and short term), to-do lists, and journal entries. The data will be stored in local storage and the user will be able to save to, or load from, a file.

Scenarios

  • How can I set goals and keep them in mind throughout the week?
  • How can I keep track of tasks for Work, Home, and Church while defining each task with a priority?
  • How can I record and store my journal entries?
